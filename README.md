# Flight Booking Database System (SQL Server)

## Project Overview
This project involves designing and implementing a relational database system for a flight booking agency. The system manages customers, flights, bookings, and related data using SQL Server.

## Features
- Entity Relationship Diagram (ERD)
- Table creation with primary and foreign key constraints
- Stored Procedures
- Views and Triggers
- Data validation and constraints
- Indexing for performance optimization
- Complex SQL queries for reporting and booking management

## Technologies Used
- Microsoft SQL Server
- T-SQL
- Relational Database Design
